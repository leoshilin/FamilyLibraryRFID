## 一、整体设计原则

1. **家庭（Family）是系统的第一业务边界**

2. **书籍信息分层设计**：

   * `book_meta`：书籍在“世界层面”的客观定义（跨家庭共享）

   * `book_item`：家庭中真实存在的一本实体书

3. **套装书是出版形态，不是库存属性**

4. **RFID 为可选能力，允许为空、可重绑定**

5. **所有关键操作必须可审计（时间 + 操作人）**

## 二、核心实体关系总览（文字 ER）

```
User ──< FamilyMember >── Family ──< Bookshelf
                                └──< BookItem >── BookMeta
                                                  └──< RFIDTag
```

## 三、用户与家庭

### 1️⃣ 用户表：user

| 字段名        | 类型                 | 中文说明           |
| ---------- | ------------------ | -------------- |
| id         | bigint PK          | 用户唯一 ID        |
| openid     | varchar(64) UNIQUE | 微信小程序 / 平台唯一标识 |
| nickname   | varchar(64)        | 用户昵称           |
| avatar_url | varchar(255)       | 用户头像 URL       |
| created_at | datetime           | 用户创建时间         |

### 2️⃣ 家庭表：family

| 字段名           | 类型          | 中文说明         |
| ------------- | ----------- | ------------ |
| id            | bigint PK   | 家庭唯一 ID      |
| name          | varchar(64) | 家庭名称         |
| owner_user_id | bigint FK   | 家庭创建者（默认管理员） |
| created_at    | datetime    | 家庭创建时间       |

### 3️⃣ 家庭成员表：family_member

| 字段名       | 类型                                  | 中文说明                 |
| --------- | ----------------------------------- | -------------------- |
| id        | bigint PK                           | 记录 ID                |
| family_id | bigint FK                           | 所属家庭 ID              |
| user_id   | bigint FK                           | 成员用户 ID              |
| role      | enum('admin','member')              | 成员角色（管理员 / 普通成员）     |
| status    | enum('pending','active','rejected') | 邀请状态（待审批 / 已加入 / 拒绝） |
| joined_at | datetime                            | 实际加入家庭时间             |

> 📌 被邀请成员扫码加入后，需管理员审批才可生效。

## 四、书架（库位）管理

### 4️⃣ 书架表：bookshelf

| 字段名         | 类型           | 中文说明         |
| ----------- | ------------ | ------------ |
| id          | bigint PK    | 书架 ID        |
| family_id   | bigint FK    | 所属家庭         |
| name        | varchar(64)  | 书架名称         |
| description | varchar(255) | 书架描述（位置说明）   |

## 五、书籍核心结构（重点）

### 5️⃣ 书籍元数据表：book_meta（跨家庭共享｜套装定义在此）

| 字段名             | 类型                               | 中文说明                  |
| --------------- | -------------------------------- | --------------------- |
| id              | bigint PK                        | 书籍元数据 ID              |
| isbn            | varchar(20)                      | ISBN（可能为空，如老书 / 内部资料） |
| title           | varchar(255)                     | 书名                    |
| subtitle        | varchar(255)                     | 副标题                   |
| authors         | varchar(255) / json              | 作者（可多作者）              |
| publisher       | varchar(255)                     | 出版社                   |
| publish_year    | varchar(10)                      | 出版年份                  |
| price           | decimal(10,2)                    | 定价（如可获取）              |
| currency           | string                    | 通货              |
| is_set          | boolean                          | 是否为套装书                |
| set_total_count | int                              | 套装共几本（未知可为空）          |
| source          | enum('system','douban','manual') | 数据来源                  |
| cover_url          | varchar(255) | 封面图片地址（云存储 URL）                  |
| cover_source          | enum('none','user_photo','official（预留）') | 封面来源                  |
| cover_updated_at      | datetime                         | 封面更新时间             |
| created_at      | datetime                         | 元数据创建时间               |
| updated_at      | datetime                         | 元数据最后更新时间             |

> 📌 套装书的“共几本”属于出版事实，应统一维护在此表中。

### 6️⃣ 家庭实体书表：book_item（库存级）

| 字段名          | 类型                         | 中文说明          |
| ------------ | -------------------------- | ------------- |
| id           | bigint PK                  | 实体书 ID        |
| family_id    | bigint FK                  | 所属家庭          |
| book_meta_id | bigint FK                  | 对应书籍元数据       |
| bookshelf_id | bigint FK                  | 当前所在书架        |
| set_index    | int NULL                   | 套装中的第几本（仅套装书） |
| edition_type | enum('original','reprint') | 原版 / 影印版      |
| status       | enum('in_stock','out')     | 在库 / 已出库      |
| rfid_tag_id  | bigint FK NULL             | 绑定的 RFID 标签   |
| created_at   | datetime                   | 入库时间          |
| updated_at   | datetime                   | 最近更新时间        |

> 📌 约束建议：
>
> * `book_meta.is_set = true` 时才允许填写 `set_index`
>
> * 同一家庭 + book_meta_id + set_index 建议唯一索引
