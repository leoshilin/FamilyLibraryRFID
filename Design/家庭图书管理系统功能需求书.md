# 家庭图书管理系统功能需求书

## 1. 项目背景与目标

随着家庭藏书数量的增加，传统人工记录方式难以满足快速查找、准确盘点和高效管理的需求。本项目拟建设一套**家庭图书管理系统**，结合 **ISBN 条码识别 + RFID 技术**，实现图书的数字化登记、精准定位、快速盘点和完整生命周期管理。

**总体目标：**

* 降低图书管理成本与复杂度

* 提升查找与盘点效率

* 支持无 RFID 与有 RFID 的混合使用场景

* 系统简洁、适合家庭或小型私人图书室使用

## 2. 用户角色定义

* **普通用户（家庭成员）**：

  * 查找图书 （家庭使用不需考虑借阅归还场景）

  * 报表（书架书籍清单，家庭书籍清单，指定期间的新入/出库书籍清单 etc.）

* **管理员（默认家庭拥有者）**：

  * 家庭成员邀请审批和删除，角色分配

  * 图书入库 / 出库

  * 图书信息修改

  * 库位管理

  * RFID 绑定与盘点

  * 系统配置与数据维护

## 3. 功能需求概述

系统功能分为以下模块：

1. 图书登记与入库管理

2. 图书信息维护

3. 图书查找与定位

4. 库存盘点

5. 库位管理

6. 图书出库管理

7. 借阅管理（建议补充）

8. 统计与报表（建议补充）

9. 系统设置与数据管理（建议补充）

## 4. 详细功能需求

### 4.1 图书登记管理（入库）

#### 4.1.1 ISBN 扫码获取信息

* 支持扫描图书 ISBN 条形码

* 系统通过在线接口获取图书信息：

  * 书名

  * 作者

  * 出版社

  * 出版时间

  * 定价

  * ISBN

  * 封面图片（如可获取）

* 在线查询失败时：

  * 提供手动录入入口

  * 支持后续补全信息

#### 4.1.2 RFID 标签写入（可选）

* 支持扫描空白 RFID 标签

* 将以下信息写入 RFID：

  * 图书唯一 ID（非直接写完整书名，避免泄露）

  * 可选：校验码 / 简要信息

* RFID 为**非必填项**：

  * 无 RFID 图书仍可仅通过 ISBN / 系统编号管理

#### 4.1.3 入库确认

* 入库时可设置：

  * 库位（可选）

  * 入库时间（系统自动）

  * 备注（如：精装、签名本等）

### 4.2 图书信息维护

* 支持编辑图书基础信息

* 支持以下附加字段：

  * 分类 / 标签（如：文学、技术、儿童）

  * 阅读状态（未读 / 在读 / 已读）

  * 收藏等级（普通 / 重要 / 珍藏）

  * 个人备注

* 支持封面图片上传或替换

### 4.3 图书查找与定位

#### 4.3.1 基础检索

* 支持模糊搜索：

  * 书名

  * 作者

  * ISBN

  * 标签 / 分类

* 支持组合筛选：

  * 是否在库

  * 是否绑定 RFID

  * 库位

#### 4.3.2 RFID 寻书模式

* 选择一本已绑定 RFID 的图书

* 启动“寻书模式”：

  * 手持 RFID 扫描枪

  * 当接近目标标签时：

    * 发出滴滴声

    * 支持声音频率 / 音量变化（距离越近越快）

* 若图书未绑定 RFID：

  * 提示仅能显示最后入库库位信息

### 4.4 库存盘点功能

#### 4.4.1 盘点模式启动

* 管理员开启盘点任务

* 选择盘点范围：

  * 全库盘点

  * 指定库位盘点

#### 4.4.2 RFID 扫描盘点

* 手持 RFID 扫描枪在房间内移动

* 每读取一个合法 RFID：

  * 扫描枪发出提示音

  * 系统记录已读取状态

#### 4.4.3 盘点结果分析

* 盘点结束后自动生成差异清单：

  * 系统在库但未扫描到（疑似丢失）

  * 扫描到但系统无记录（异常标签）

* 支持对差异项进行人工确认与处理

* 生成盘点报告（时间、数量、差异）

### 4.5 库位管理

* 支持创建 / 编辑 / 停用库位

* 库位属性包括：

  * 库位名称（如：书房 A 架 2 层）

  * 描述

  * 最大可存放数量

* 显示库位使用情况：

  * 已存放数量

  * 剩余容量

* 入库 / 移库时可选择库位

### 4.6 图书出库管理

* 支持选择单本或多本图书出库

* 出库原因可选：

  * 盘点损失

  * 出售

  * 捐赠

  * 废弃

  * 其他（自定义）

* 出库后状态：

  * 默认不删除数据，仅标记为“已出库”

  * 保留历史记录与原因

## 5. 建议补充功能

### 5.1 借阅管理（家庭内部）

* 记录借阅人

* 借出 / 归还时间

* 借阅状态提醒

### 5.2 统计与报表

* 藏书总量统计

* 分类 / 作者 / 出版社分布

* 年度入库 / 出库趋势

* RFID 覆盖率统计

### 5.3 数据管理与安全

* 数据本地备份 / 云备份

* 支持数据导出（CSV / Excel）

* RFID 与图书信息解耦，避免标签泄露隐私

### 5.4 系统配置

* ISBN 查询接口配置

* RFID 设备参数配置

* 声音提示开关与音量设置

## 6. 非功能性需求

* **易用性**：界面简洁，适合非专业用户

* **可靠性**：盘点与数据写入需有失败重试机制

* **扩展性**：支持未来扩展到更多物品管理

* **离线能力**：基础功能在断网情况下可用（ISBN 查询除外）

## 7. 未来扩展方向（可选）

* 手机端 App（蓝牙 RFID）

* 图书推荐与阅读统计

* 与电子书平台联动

* 家庭共享与权限控制
